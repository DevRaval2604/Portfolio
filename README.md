# Dev Raval – Portfolio (Next.js + Tailwind)

This project recreates the Dev Raval portfolio UI using **Next.js (App Router)**, **Tailwind CSS**, and **Framer Motion** with a focus on pixel-perfect layout, dark theme, gradients, and smooth animations.

## Commands

- `npm install` – install dependencies  
- `npm run dev` – start the development server at `http://localhost:3000`  
- `npm run build` – build for production  
- `npm run start` – run the production build
